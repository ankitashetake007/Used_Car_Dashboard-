# Used_Car_Dashboard-

# Project Overview

This project focuses on analyzing Used Car Market Data using Power BI to uncover insights related to pricing trends, car conditions, brand performance, mileage patterns, and factors influencing resale value.

The dashboard helps users make data-driven decisions—whether they are buyers, sellers, or market analysts.

# Objectives

- Understand how car features (year, mileage, fuel type, condition, brand) influence resale price.

- Identify top brands in different segments.

- Analyze demand trends based on price categories.

- Provide an interactive dashboard for quick comparison and exploration. 

# Key Insights 

- Price distribution across different car categories

- Mileage vs. Price correlation

- Best-selling and high-value brands

- Condition-wise average pricing  

- Year-of-manufacture impact on resale value

- Fuel type–wise performance comparison

# Tools & Technologies Used

- Power BI Desktop

- DAX measures

- Data Cleaning & Transformation (Power Query)

- Data Visualization

- Excel datasets

# Conclusion

This analysis helps understand what influences used car prices the most and provides actionable insights for buyers, sellers, and dealership decision-making.
